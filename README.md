## 🚀 Currently Building

**AirFlow Station V2.0.1** — A 100% off-grid, filterless air purification unit for highway medians.
Powered by bifacial solar + traffic-wake wind energy. ESP-based. No filters. No grid dependency.

- 🏫 Pre-incubated at **[TISC, IIT Jodhpur](https://tisc.iitj.ac.in/)** (Technology Innovation & Startup Centre)

- 🔬 Phase 1 prototype in active development – ESP validation + energy harvesting
